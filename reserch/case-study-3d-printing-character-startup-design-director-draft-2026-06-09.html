Executive summary

The startup needed a printable character product system, not just more models.

The team wanted to produce different collectible characters, but early assets were inconsistent: some looked strong on screen but failed or required heavy repair before printing. Lesly created the design-direction and production-readiness plan so characters could move from concept to printable files with fewer surprises.

The problem

Beautiful digital characters were not automatically printable products.

Printability gap

Screen-ready models failed production checks.

Some characters had fragile details, unsupported silhouettes, thin accessories, non-watertight geometry, or features that looked good in render but broke in resin or FDM workflows.

No product system

Each character was treated as a one-off.

The startup lacked a shared style guide, scale system, accessory rules, pose constraints, base standards, and reusable library for future character families.

Slow handoff

Print prep relied on repeated repair cycles.

Modelers and printing operators spent time fixing supports, thickness, hollowing, drainage, orientation, and file exports after design decisions had already been made.

Design direction system

Lesly defined the rules that make characters printable, collectible, and repeatable.

Hero scale

Large figurines with base, stronger silhouette, and hollowing review.

Mini scale

Small characters with simplified details and stronger limbs/accessories.

Accessory rules

Weapons, horns, hair, capes, and props checked for thickness and support.

Base system

Reusable bases for stability, story, packaging, and line consistency.

Lesly's design-director contribution

  • Defined the target audience and collectible character families.
  • Created style rules for silhouette, face read, pose, accessories, and base design.
  • Translated 3D character expertise into printability standards.
  • Specified the review gates before sculpt, before export, and before print.
  • Trained the team to evaluate models through design quality and physical production constraints.

Production workflow

The workflow moved printability earlier in the design process.

Product brief

Define character family, scale, audience, pose, base, and intended print process.

Style board

Lock visual language: silhouette, proportions, expression, accessories, and collectible appeal.

Print rules

Check minimum thickness, overhangs, supports, hollowing, drainage, and fragile details.

Model review

Review watertight geometry, scale, split parts, base fit, and file-export readiness.

Slicer test

Validate orientation, supports, island risks, material use, and estimated print time.

Handoff

Deliver STL/3MF/OBJ package, print notes, renders, thumbnails, and product metadata.
