Website copy upgrade review

Keep Lesly grounded as a 3D Character Artist, then make her direction judgment visible.

This review compares the live homepage copy with a stronger, field-native version. The goal is sharper public copy: character-first, production-aware, human-written, and careful about AI or director-level claims.

Primary recommendation

Do not rename Lesly as Creative Director. Position her as a 3D Character Artist who helps define character visual direction, asset scope, pipeline, and handoff before the build gets expensive.

Executive verdict

The site is close. The copy now needs clearer hierarchy.

The current homepage already names character assets, production planning, and handoff. The next upgrade should make the first screen less like a list of asset categories and more like a buyer promise: Lesly turns rough character inputs into usable 3D character assets with a defined production path.

Keep

3D Character Artist

This is the safest title for recruiters, game teams, and freelance buyers. It matches current character-art hiring language and avoids senior leadership overclaiming.

Elevate

Direction as the second visible lane

Put Character Direction & Pipeline Strategy directly after character production. This shows Lesly defines the character set, identifies the production path, improves the workflow, and plans the handoff instead of only making requested 3D assets.

Demote

AI as identity

AI can be mentioned as reference exploration or workflow support. It should not lead the page, and it should not replace the craft language: sculpting, modeling, retopo, UVs, textures, and final asset delivery.

Best public positioning:

Lesly is a 3D Character Artist who creates game-ready characters, mascots, and avatar assets, then helps define the character direction, asset set, production pipeline, and handoff path around them.

Internet research findings

Current professional wording supports a craft-first, proof-backed direction.

Recent job descriptions and portfolio guidance repeat the same signal: a character artist is judged by craft, pipeline awareness, real-time constraints, and proof materials. Director words are valid only when they describe standards, guidance, reviews, or pipeline ownership that can be shown.

AI wording
AI references and AI-assisted reference exploration are safer than AI Artist, AI-Assisted Visual Designer, or AI-first consulting. The final deliverable should stay human-directed and 3D-production focused.

Studio-facing takeaway

Studios want proof that a character can move from concept into a real production pipeline. The copy should name topology, UVs, textures, export notes, and real-time requirements only where Lesly can show supporting material.

Client-facing takeaway

Founders and creators need help turning a loose character idea into a brief, style direction, scoped package, and handoff plan. This is the premium lane for the site.

Humanizer pass notes

The replacement copy should sound direct, not inflated.

The humanizer pass removes formulaic AI writing patterns and portfolio cliches while preserving the core offer.

Remove or reduce

  • Vague vision claims without proof.
  • Promotional adjectives such as stunning, captivating, premium, and world-class.
  • Rule-of-three rhythm that turns every sentence into a slogan.
  • AI-first phrasing that makes the workflow sound automated.
  • Director titles when the evidence only supports direction support.

Use instead

  • Plain craft nouns: sculpting, modeling, retopo, UVs, textures, renders.
  • Buyer nouns: sketch, moodboard, AI reference, brief, platform, file package.
  • Planning nouns: visual direction, asset scope, review checkpoint, handoff note.
  • Specific CTAs: Start a Project, View Work, Ask for Direction and Scope.
  • First-person or Lesly-centered sentences that state the work cleanly.
Humanized line:

If your character idea is still rough, Lesly can help define the look, use case, files, and production path before modeling begins.
