# Research Folder

This folder saves the strategy, chat history, design decisions, and visual asset notes from the Lesly 3D Artist website project.

## Contents

- `chat-history.md` — project conversation summary and decision trail
- `platform-strategy.md` — freelance platform and service-positioning strategy
- `website-information-architecture.md` — conversion-focused website structure
- `visual-assets-checklist.md` — launch asset checklist and service-specific visuals
- `design-system-spec.md` — reusable colors, typography, grid, and component system
- `visuals/visual-index.md` — index of generated visuals and user-provided references
- `visuals/visual-gallery.html` — compact visual contact sheet preview
