# Visual Assets Checklist — Lesly Website and Freelance Profiles

## Minimum launch set

1. Hero render
2. 6 strong portfolio thumbnails
3. 3 detailed case-study-style portfolio breakdowns
4. 3 before/after visuals
5. 1 process diagram
6. 5 service card visuals
7. 1 profile portrait
8. 1 social/banner image
9. 1 project inquiry CTA visual
10. 1 PDF portfolio cover

---

## Must-have visual assets

| Asset | Where it appears | What it should show |
|---|---|---|
| Hero character render | Homepage top section | Strongest 3D character, premium lighting, clean background |
| Portfolio cover images | Work / Portfolio page | 6–9 best pieces with strong thumbnails |
| Character turnarounds | Portfolio case studies | Front, side, and back views |
| Close-up detail renders | Case studies / services | Face, hands, clothing, accessories, textures |
| Before → After visuals | Homepage + Services | Sketch / AI reference / rough idea → final 3D character |
| Process breakdown images | Process page / case studies | Concept → sculpt → model → texture → final render |
| Wireframe / topology screenshots | Game-ready service page | Production-quality 3D workflow |
| Texture/material sheets | Character case studies | PBR textures, UVs, materials |
| Pose render set | Portfolio and service pages | 2–4 poses per strong character |
| Transparent PNG cutouts | Homepage, service cards, socials | Flexible character cutouts |
| Service card visuals | Services section | One image per service |
| Artist portrait | About page, LinkedIn, Upwork, Contra | Professional and creative profile image |
| Logo / wordmark | Header, footer, PDF | Simple Lesly personal brand mark |
| Social banner image | LinkedIn, ArtStation, Behance, Contra | Wide banner with tagline and best work |
| Inquiry CTA visual | Start a Project page | Friendly visual that supports conversion |

## Assets by service

### Concept / AI Reference → 3D Character

- AI reference or sketch placeholder
- Refined character concept
- Sculpt progress image
- Final 3D render
- Before/after comparison
- Visual pipeline graphic

### Game-Ready Character Package

- Final character render
- Low-poly model view
- Wireframe overlay
- UV layout screenshot
- Texture sheet
- Unity / Unreal mockup
- File delivery graphic: FBX, OBJ, BLEND, PNG textures

### Avatar / VTuber / Streamer Character

- Front-facing avatar render
- Expression sheet
- Outfit variations
- Accessories
- Profile image mockup
- Streaming overlay mockup
- Optional rig-ready pose preview

### Brand Mascot / Product Character

- Mascot hero render
- Expression set
- Pose set
- Brand color variation
- Website/app mockup
- Social media post mockup
- Sticker/emote-style versions

### Roblox / UGC / Stylized Assets

- Accessory renders
- Clothing/item previews
- Simple character wearing the asset
- Platform-style thumbnail
- Low-poly/stylized model views
- Variation set

### AI-Assisted Visual Development

- Moodboard
- Style exploration grid
- Multiple concept variations
- Final selected direction
- Human-refined final design
- Exploration-to-final collage

### Art Direction / Consulting

- Character style guide sample
- Pipeline diagram
- Moodboard example
- Visual checklist example
- Before/after art direction improvement
- Sample slide or board showing how direction is explained
