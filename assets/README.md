# Final Site Asset Folders

These folders contain portfolio media, source mesh packages, and real site assets created for Lesly's portfolio.

## Folders

- `portfolio/` - homepage hero and work-card thumbnails.
- `process/` - case-study renders, starting points, process images, technical proof, and deliverables proof.
- `3d/` - source mesh packages, material libraries, manifests, and technical sheets created for case-study proof.
- `profile/` - Lesly portrait/profile images and studio/process visuals.
- `brand/` - brand SVGs used by the site.

## Rules

- Generated assets created for this portfolio are real website assets. Inspect each file individually for design quality, anatomy, material read, lighting, composition, and fit with the service it represents.
- Do not invent external client names, testimonials, shipped-game credits, engine integrations, production metrics, or source-file details that are not in the repo.
- Do not copy `../../reserch/visuals/image*.png` here as portfolio proof.
- Repo-created source mesh packages must include inspectable source files, material assignments, and a manifest before being used as production evidence.
