## Reconstruction Priorities

1. Preserve one skull, eye spacing, nose projection, lip volume, jaw width, ears, neck length, and shoulder slope across every view.
2. Establish cranial mass, bust silhouette, neck/shoulder transition, high braided bun, front locks, and collar before tertiary detail.
3. Build the hair as readable grouped masses before strand detail; keep the diamond forehead ornament, earrings, and chest emblem aligned and structurally separate.
4. Validate a true profile before retopology. Check forehead-to-nose rhythm, lips, chin projection, jaw angle, ear placement, bun depth, and neck connection.
5. Plan deformation-aware loops only after the matching high-poly is stable. Prioritize eyelids, mouth corners, nasolabial/cheek transition, jaw hinge, ears, hairline, and neck.
6. Capture the real high-poly, low-poly wireframe, loop decisions, density transitions, packed UVs, bakes, neutral material checks, source-tool context, and exports before adding technical completion claims.

## Generation Prompt Set

All three images were created with the built-in image-generation tool using the earlier clay bust as the identity reference.

### Hero

Create one premium resolved warm-gray digital-clay bust of the same young woman in a calm three-quarter view. Preserve her facial identity, braided high bun, loose front strands, diamond forehead ornament, earrings, high collar, and chest emblem. Use neutral form-inspection light with a restrained warm rim on a seamless charcoal background. Keep the full hair, shoulders, collar, and ornament visible. Exclude text, interface elements, topology, wireframe, logos, and decorative infographic elements.

### Broad-to-Fine Sculpt Development

Create three coordinated bust stages of the same character: primary-form blockout, secondary-form refinement, and resolved neutral clay sculpt. Keep one identity and one continuous studio presentation. Make the hierarchy readable through anatomy and surface resolution rather than labels, boxes, arrows, bubbles, or software chrome. The first stage is a form visualization, not a topology claim.

### Form and Deformation Targets

Create four coordinated clay busts of the same character: neutral front, true left profile, controlled closed-mouth smile, and controlled jaw-open expression. Preserve one skull, face, hair, ornament, earrings, collar, and chest emblem. Keep facial planes, eyelids, lips, jaw hinge, cheek volume, ear placement, and neck transitions visible under neutral inspection light. Exclude topology lines, wireframe, interface panels, labels, and rigging claims.
