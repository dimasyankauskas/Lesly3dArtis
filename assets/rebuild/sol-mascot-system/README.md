# Sol Mascot Identity System

This folder contains approved visual direction for rebuilding Sol as one consistent 3D mascot. The images are public portfolio assets and a recreation target, not captures from a completed production model.

## Presentation set

- `../sol-mascot/sol-hero-4x5-v1.webp` — canonical hero and primary identity reference.
- `identity-views-16x9-v1.webp` — coordinated front, three-quarter, profile, and rear presentation views.
- `pose-expression-16x9-v1.webp` — greeting, guide, delighted, and thoughtful pose-and-expression targets.

## Reconstruction priorities

1. Preserve the fennec muzzle, oversized ears, amber eyes, and compact body proportions.
2. Keep exactly three terracotta forehead markings and one geometric band on one tail.
3. Build compact four-digit animal paws; do not substitute human hands or mitten forms.
4. Preserve the ivory tunic, terracotta scarf and cape, leather belt and pouch, aged-bronze protection, wrapped boots, and leaf-topped staff family.
5. Match the silhouette and material hierarchy before expanding the pose or expression range.

## Evidence boundary

The package defines identity, view consistency, pose, expression, costume, prop, palette, and surface direction. It does not prove a matching 3D model, measured orthographic turnaround, topology, UVs, textures, rig, deformation, exports, or a deployed brand application. Those require captures and tests from the future production asset.

## Source masters

The lossless generation masters are retained under `source/`. Public pages use the compressed WebP assets in this directory.
