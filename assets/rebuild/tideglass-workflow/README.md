# Tideglass Courier Visual Workflow

Public reconstruction-direction media for `ai-visual-pipeline`.

- `hero-4x3-v1.webp` — physical art-direction table with one resolved Tideglass Courier target, coordinated sketches, a neutral maquette, and material swatches.
- `direction-sequence-16x9-v1.webp` — coordinated sketch, neutral-form, and material-direction sequence.
- `review-handoff-16x9-v1.webp` — face, staff grip, costume construction, rear silhouette, material family, and blank handoff-folder review direction.
- `source/` — lossless generated masters retained for future non-destructive derivatives.

These images define visual-workflow and character-reconstruction direction. They are not captures from a completed production model, software project, topology, UV, texture, rig, engine, source-file, approval-log, or delivered handoff package.

Built-in image generation was used for all three assets. The hero became the identity reference for the two supporting images so the same short-haired courier, ivory-and-sea-green coat, copper shoulder guard, satchel, boots, and navigation staff remain coherent.
