# Coastal Residence Visualization Direction

Public reconstruction-direction media for `santa-cruz-ai-visual-pipeline`.

- `hero-4x3-v1.webp` — resolved coastal-residence direction presented with physical elevation, context, camera, and finish references.
- `material-options-16x9-v1.webp` — one coordinated camera comparing three restrained exterior finish families.
- `context-lighting-16x9-v1.webp` — site-context, neutral-daylight, and blue-hour presentation direction.
- `source/` — lossless generated masters retained for non-destructive derivatives.

These images define a fictional coastal-residence visualization study. They do not document a verified Santa Cruz address, client, site, drawings, constructed project, matching production scene, construction decision, implemented local-AI system, approval, budget, schedule, savings, or measured outcome.

Built-in image generation produced all three assets. The hero became the visual reference for both supporting images so the low-slope roof, broad glazing, stone piers, warm wood, charcoal metal, stepped approach, boulder landscape, and native coastal planting remain coherent.
