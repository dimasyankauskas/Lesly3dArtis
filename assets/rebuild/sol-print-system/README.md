# Sol Collectible Build Direction

This folder contains visual direction for rebuilding Sol as a collectible character. The images are public portfolio assets and 3D recreation targets, not captures from a printable production model or a physical print.

Attribution source truth recorded 2026-07-19: Lesly authored the Sol Collectible website case and portfolio presentation from zero to one. Public copy may identify her as the portfolio case author and credit the case structure, color-and-neutral presentation, visual-grouping study, contact-view sequence, public narrative, and website presentation. That attribution does not establish authorship of the inherited Sol identity, a matching source model or sculpt, print engineering, actual parts or joints, slicer work, manufacture, a physical print, painting, delivery, or outcomes.

## Presentation set

- `../sol-print/sol-collectible-4x3-v2.webp` — paired color and neutral-clay hero direction.
- `assembly-direction-16x9-v2.webp` — assembled maquette plus proposed head, cape/scarf, tail, body, paw, staff, holder, and base groups; the apparent openings and separations are generated visual planning cues, not cuts, sockets, parts, joints, or source geometry. Keep this boundary attached whenever the image is reused.
- `contact-views-16x9-v1.webp` — coordinated front-three-quarter, profile, and rear-three-quarter maquettes for silhouette, contact, cape, tail, staff-holder, and base review.

## Reconstruction priorities

1. Preserve Sol’s fennec anatomy, exactly three forehead marks, one patterned tail, compact animal paws, costume, staff, and base language.
2. Rebuild the approved maquette before deciding actual part boundaries, joints, keys, tolerances, or manufacturing scale.
3. Review the ears, staff, tail, cape edges, paws, small bronze forms, and base footprint as structural risk zones.
4. Keep the staff seated in its holder and the character grounded through both feet and a stable tiered base.
5. Validate every engineering decision on the matching production model and with the intended print process.

## Evidence boundary

The package defines collectible form, visual part-group direction, silhouette, contact, prop, and base priorities. It does not prove a production mesh, final part separation, joints, keys, dimensions, wall thickness, watertightness, hollowing, drainage, supports, slicer settings, source files, exports, assembly, or a successful physical print. Those require real model inspection, slicer evidence, and test-print photographs.
