## Direction decision

The public story is about visible costume hierarchy, not completed garment production. Ivory foundation clothing and charcoal trousers establish the base read; plum wrap tailoring and split coat tails add the primary silhouette; dark leather harness and waist structure organize the overlap; asymmetric aged-gold protection, fasteners, brooch, boots, and the swept-hilt rapier complete the final visual target. The front/back sheet exposes rear silhouette and routing questions that the hero hides, while the layer sheet separates each visual contribution.
