# Lesly V2 Staging Site

Belleza-based V2 staging site for Lesly's senior 3D Character Artist portfolio.

Current status: browser-reviewable and visually populated with original generated V2 concept assets. It is not final public launch until the contact destination, public profile decisions, generated-asset approval, and any desired real source 3D production proof are confirmed.

Preview from the repo root:

```bash
python3 -m http.server 8000 --bind 127.0.0.1
```

Open:

```text
http://127.0.0.1:8000/v2/
http://127.0.0.1:8000/v2/case-study.html
```

Use `LAUNCH_CHECKLIST.md` for exact launch blockers, `ASSET_INTAKE.md` to record approved source filenames, project facts, contact details, and replacement status before editing HTML, and `ENVIRONMENT_STRATEGY.md` for the character-first lighting, camera, stage, and viewer standard.

Do not use `../reserch/visuals/image*.png` as portfolio proof. Those files are design/mockup references, not verified Lesly work.
