# V2 Asset Folders

These folders contain original generated V2 concept assets plus the replacement structure for future approved Lesly work.

## Folders

- `portfolio/` - homepage hero and work-card thumbnails.
- `process/` - case-study renders, starting points, process images, technical proof, and deliverables proof.
- `profile/` - approved Lesly portrait/profile images or explicitly labeled studio/process visuals.
- `brand/` - V2 brand SVGs used by the site.

## Rules

- Generated originals must be labeled as generated/site-created concept visuals anywhere they could be read as portfolio proof.
- Do not claim generated images are client work, shipped work, source 3D proof, topology, UV, texture, engine, or handoff evidence.
- Do not copy `../../reserch/visuals/image*.png` here as portfolio proof.
- Use `../ASSET_INTAKE.md` to record source, target path, status, and factual limits before updating HTML.
- Real Lesly proof can replace generated concept assets only after the source files and project facts are recorded.
