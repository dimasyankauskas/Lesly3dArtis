# Skill: Private Commission Portal

Use this skill when designing the VIP client portal concept.

## Features
- private mobile page
- visual progress updates
- approval status
- image comments / pins
- material studies
- video updates
- final reveal archive

## Business value
Trust, premium experience, fewer scattered messages, faster decisions, stronger client confidence.

## Pitch-demo boundary

For the S-RANCH proposal, present the portal as a prototype / future workflow concept unless production integration has been scoped.

Safe wording:

- "Private mobile approval page"
- "Tap-to-comment prototype"
- "Can be connected to email, CRM, dashboard, Slack, or project management in a production phase"

Avoid implying current production support for:

- login / private token authentication
- live CRM, Slack, email, SMS, or project-management routing
- approval database or staff dashboard
- legally binding approval records

## Final proposal rule

Lead with the Visual Strategy Pilot first. Keep the Premium Private Commission Portal as an optional next stage after the pilot proves value.
