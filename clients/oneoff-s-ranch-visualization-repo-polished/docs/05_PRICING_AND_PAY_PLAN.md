# 05 — Pricing and Pay Plan

These ranges are strategic planning estimates. They should be adjusted based on scope, deadlines, number of revisions, required accuracy, ownership, and whether One Off wants a one-time project or ongoing role.

## Package 1 — Visual Strategy Pilot
**$2,500–$5,000**

Best for proving value on one project.

Includes:
- project review
- technical interpretation notes
- 1–2 concept visuals
- one small client-facing presentation page
- recommendation for next stage

## Package 2 — Project Visualization Package
**$5,000–$12,000**

Best for a live VIP project where client approval matters.

Includes:
- hero visual
- closed/open or partial-open view
- detail view
- material/mood board
- short presentation page
- one revision cycle

## Package 3 — Premium Private Commission Portal
**$12,000–$25,000+**

Best for high-value commissions where the experience itself should feel white-glove.

Includes:
- private mobile-first project page
- multiple visuals / videos
- tap-to-comment review prototype or production-ready handoff
- progress timeline
- approval notes
- final reveal archive

## Package 4 — Fractional Visualization & Experience Director Retainer
**$6,000–$15,000+/month**

Best for recurring high-value projects.

Includes:
- embedded visualization strategy
- visual standards
- project intake system
- client portal templates
- asset library direction
- visual QA
- pitch and client-approval support

## Full-time direction
If this becomes employment, compensation should be benchmarked against director-level visualization / art direction / creative technology roles, not junior 3D artist rates.

## Pricing principle
Do not price only for “making an image.” Price for:

- reducing client uncertainty
- improving approvals
- supporting higher perceived value
- saving staff time
- creating reusable client and marketing assets
- designing a repeatable visual system
