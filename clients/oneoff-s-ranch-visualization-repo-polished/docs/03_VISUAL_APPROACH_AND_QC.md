# 03 — Visual Approach and Quality Control

## Goal
Create visuals that are beautiful enough for a VIP client but accurate enough to be credible to One Off and the fabricator.

## Required visual set
1. Hero closed view: finished built-in bar wall.
2. Right 45-degree closed view: depth and integration.
3. Left-pair-open view: only the left bifold pair folded and parked left.
4. Both-sides-open view: left pair parked left, right pair parked right.
5. Mechanism logic board: closed → fold → slide → open.
6. Detail close-up: rail / shoulder bolt / blackened metal frame.
7. Detail close-up: upper glass/infill direction and woodwork if the infill is confirmed.
8. Private client portal mockup.

## QC checklist for every render
- Four lower panels total? Yes/no.
- Left pair = exactly two panels? Yes/no.
- Right pair = exactly two panels? Yes/no.
- If left open only, right pair remains closed? Yes/no.
- If both open, left pair parked left and right pair parked right? Yes/no.
- No full room/kitchen behind the opening? Yes/no.
- Recessed niche depth feels shallow-to-moderate? Yes/no.
- Upper arched glass area preserved without overclaiming final infill? Yes/no.
- Side shelving and lower cabinetry consistent? Yes/no.
- Warm ranch/luxury atmosphere preserved? Yes/no.

## Visual language
Use:

- warm walnut / oak / rich wood
- blackened steel or dark bronze frames
- glass/infill only if confirmed; otherwise keep it subtle or omit lattice
- carved millwork
- dark stone counter
- understated warm lighting
- Montana ranch / lodge / private residence atmosphere

Avoid:

- generic modern white aluminum bifold doors
- freestanding cabinet object
- excessive cinematic fantasy lighting
- generic luxury hotel bar
- obvious AI distortions
